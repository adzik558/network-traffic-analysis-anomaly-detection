# Network Traffic Analysis & Anomaly Detection (R, RShiny)

## Overview
Analiza ruchu sieciowego (Wireshark export, ~400k pakietów). Celem: analiza protokołów, statystyki pakietów i wykrycie anomalii.

## Technologies
R, tidyverse, ggplot2, plotly, shiny, tsoutliers, isotree (isolation forest)
